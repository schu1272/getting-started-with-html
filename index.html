The basics of the internet

The internet is formed within the following parts

1. Requests are sent through a HTTP which stands for Hypertext Transfer Protocal
2. These requests use the worlds largest computer network called the Internet
3. The request are received by Servers which are basically computers optimized to store files
4. Browsers receive and interprate the files and display the results to the users computer


Webpage Foundation

Websites on their own are just plaintext. It does not take a lot to make a simple website
However developers use HTML Or Hypertext Markup Language to add design to a website and format content on the website.
HTML has the following parts:
- Tags
- Elements

HTML elemnts are made up of tags. Elements being and end with a tag.
